به زودی تمام پلاگین ها فارسی میشه 

Setup
Clone m_m_F !

cd $HOME
git clone https://github.com/mohammadmahdi001/m_m_f_robot
cd m_m_f_robot
install it!

chmod +x install.sh
./install.sh
Manual installation

Add your bot token in config.moon

telegram_api_key: "Your bot token here"
Add your telegram id to admins table in config.moon

admins: {
    150289672
    213861807
    --your id
  }
Run it !

chmod +x run.sh
./run.sh
